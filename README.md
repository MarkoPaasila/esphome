# esphome
Useful configurations and experimental external components for ESPHome

## External components

- **hp_ekf** – Custom component (HP-EKF). See `components/hp_ekf/README.md` for usage.
- **hp_ukf** – Custom component (HP-UKF). See `components/hp_ukf/README.md` for usage.
- **defrost-fooler** - Custom component (mitsurunner). See `components/defrost-fooler/README.md` for usage.

## yaml config snippets

- **ddf** - Dynamic Deadband lambda sensor filter. Explanation in `ddf.yaml`.
- **drdf** - Dynamic Reversal based Deadband lambda sensor filter. Explanation in `drdf.yaml`.
